⚙️ 同步设置

配置 GitHub 仓库实现 iPad / 电脑 / Notion 之间数据实时同步。

🔑 第一步：获取 GitHub Token
1. 打开 github.com/settings/tokens
2. 点 Generate new token → Generate new token (classic)
3. Note 填 股票记录，过期选 No expiration
4. 勾选 ☑️ repo（只需这一个权限）
5. 拉到底部 → 点 Generate token
6. 复制那串字符（只显示一次！）→ 粘贴到下方

📖 使用帮助

📱 iPad / iPhone 使用：部署到 GitHub Pages → Safari 打开 → 分享 →「添加到主屏幕」

🔄 跨设备同步：在「设置」中配置 GitHub → iPad 和电脑修改互相同步

🌐 嵌入 Notion：部署后，在 Notion 中 /embed → 粘贴 URL
